Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

la Web, se originó en 1989 como un proyecto del científico británico Tim Berners-Lee, quien trabajaba en el CERN (Centro Europeo para la Investigación Nuclear) en Ginebra, Suiza

1. Web 2.0

La Web 2.0, también conocida como Web Social, es una generación de sitios web que permiten a los usuarios interactuar y colaborar de forma activa: Los usuarios pueden crear, modificar, editar, borrar y clasificar contenidos. Los sitios web son dinámicos e interactivos, y se pueden actualizar con frecuencia.

2. Web 3.0 3

La Web 3.0 es la tercera generación de internet y se caracteriza por ser descentralizada, segura y personalizada. Se basa en tecnologías como la inteligencia artificial, el blockchain y los contratos inteligentes para ofrecer una experiencia digital mejorada.

3. La W3C y los estándares

El World Wide Web Consortium (W3C) es una organización internacional que crea estándares para la World Wide Web. Los estándares web W3C son protocolos, lenguajes y tecnologías que tienen como objetivo: Mejorar el rendimiento de la web Mejorar la funcionalidad operativa para los navegadores y usuarios Lograr la interoperabilidad web, es decir, que las tecnologías web sean compatibles entre sí

3. HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

4. XHTML

XHTML (eXtensible HyperText Markup Language) es un lenguaje de marcado de hipertexto que se basa en XML y se utiliza para crear y mostrar páginas web. Es una reformulación de HTML 4 como una aplicación XML y se caracteriza por ser más estricto a nivel técnico que HTML. Algunas de las características de XHTML son XHTML es más estricto que HTML en cuanto a sintaxis y estructura. XHTML es más fácil para hacer cambios o buscar errores. XHTML es compatible con aplicaciones de usuario HTML existentes. XHTML se puede ver, editar y validar con herramientas XML estándar. XHTML distingue entre mayúsculas y minúsculas, por lo que todas las etiquetas estándar deben escribirse en minúsculas. XHTML 1.0 especifica tres tipos de documento XML: Strict, Transitional, y Frameset. XHTML Básico es una base común que se puede extender con módulos adicionales. III. Otros lenguajes relacionados al desarrollo web

1. CSS/CSS3

CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para definir el diseño visual de páginas web e interfaces de usuario, CSS3 es la versión sucesora de CSS, que se comenzó a definir a partir del año 2005. CSS3 ofrece una gran variedad de opciones para las necesidades del diseño web actual, como: Sombras para textos y cajas Bordes redondeados Color con transparencia (RGBA y HSLA) Nuevas opciones para imágenes de fondo Transiciones Animaciones Cajas flexibles Grillas Separación con múltiples columnas

2. JavaScript

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas. Se utiliza para agregar dinamismo y interacción a las páginas web, por ejemplo, para actualizar fuentes de redes sociales, mostrar animaciones o mapas interactivos. JavaScript es uno de los tres lenguajes de desarrollo web fundamentales para crear sitios web modernos y funcionales, junto con HTML y CSS. HTML define la estructura y el contenido de la página web, mientras que CSS se encarga del diseño y la presentación

3. Ajax

AJAX permite a las aplicaciones web validar información específica en formularios antes de que los usuarios los envíen. Por ejemplo, cuando un nuevo usuario crea una cuenta, la página web puede verificar automáticamente si hay un nombre de usuario disponible antes de que el usuario pase a la siguiente sección.

Tema 3: Estructura De Una Página Web

I. Jerarquía de directorio de un proyecto web

La jerarquía de directorios de un proyecto web es la organización de los archivos y carpetas de un sitio web en una estructura de árbol invertido: En la cima de la jerarquía se encuentra el directorio raíz, que es la carpeta principal y contiene toda la información del sitio web. A partir del directorio raíz se crean las ramas, que son los subdirectorios y los niveles de clasificación. Cada directorio puede tener uno o más subdirectorios. Para distinguir los niveles de directorio se utiliza el carácter "/". En un sitio web, el directorio raíz suele llamarse public_html, htdocs, www o wwwroot. Es importante porque es donde se almacena el archivo robots.txt, que es necesario para la optimización de los buscadores. La jerarquía de directorios es una forma de clasificar la información que facilita el acceso a los archivos y carpetas.

II. Elementos de estructura para el diseño de una página Web: Concepto

Los elementos de estructura para el diseño de una página web son las tres secciones que la componen: la cabecera, el cuerpo y el pie de página: Cabecera o header: Es la parte superior de la página y contiene información básica de la empresa, como el logo, el menú de navegación, el cuadro de búsqueda y una descripción de la web. Cuerpo o body: Es la sección donde se muestra el contenido principal de la página. Pie de página o footer: Es la parte inferior de la página y contiene información de contacto, botones de redes sociales, el logo de la empresa y un enlace a la política de privacidad. Además de estas secciones, otros elementos que se pueden incluir en una página web son: Título de la página Imagen principal o de portada Barra lateral Contenido visual, como imágenes y videos Buscador Llamadas a la acción (CTA) Formularios Un buen diseño web debe ser atractivo, facilitar la navegación y hacer que los visitantes disfruten su tiempo en el sitio. Para ello, se pueden utilizar elementos gráficos y textuales, como colores, tipografías, imágenes y videos.
